# Graphicfolio

## Overview

Graphicfolio Landing page

## Built with

+ Sass
+ JavaScript

## Development

1- Clone the repository and change directory.
```
git clone https://github.com/mujahedjs/graphicfolio.git
cd graphicfolio
```
2- Install npm dependencies
```
npm install -g sass
```
3- Run app locally
```
sass --watch sass/input.scss output.css
```
